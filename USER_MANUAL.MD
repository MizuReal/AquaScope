
# User Manual

## Mobile App

### Overview
The mobile app provides water quality analysis, prediction history, community forum, and user profile management. Main screens are accessible via the bottom tab bar.

### Main Features

- **Home**: Quick access to water analysis and recent activity.
- **Data Input**: Enter water sample data manually or via camera (OCR). Submit samples for analysis.
- **Container Analysis**: Scan containers for water samples, view/manage scan history, analyze results and risk levels.
- **Analytics**: Visualize water quality trends with charts. Track historical data and risk assessments.
- **Prediction History**: Review all previous water sample predictions with detailed results.
- **Community Forum**: Participate in discussions, ask questions, share results, and receive notifications.
- **Profile**: Manage user profile, avatar, password, and personal information.

**Navigation:**
Use the bottom tab bar to switch between main features. Some screens (e.g., Data Input, Profile) are accessible via navigation actions.

## Web App (web-react)

### For Users

- **Home**: Welcome page with project overview and navigation.
- **Dashboard**: Central hub for accessing scans, analytics, and community features.
- **Scans**: View all water sample predictions linked to your account. Access detailed sample history.
- **Analytics**: Visualize water quality data and trends. Export analytics as PDF.
- **Community**: Join discussions, post questions, interact with other users, and receive forum notifications.
- **Profile**: Manage your account details, avatar, and password.

### For Admins

- **Admin Dashboard**: Overview of system status and recent activity.
- **Users Management**: View, search, and manage user accounts. Deactivate users and send notifications.
- **Reports**: Review and manage user reports and flagged content.
- **Forum Management**: Moderate forum posts and threads. Manage categories and user interactions.
- **Analytics (Admin)**: Access advanced analytics, export data, and monitor system-wide trends and usage.

**Note:** All features require authentication. Some admin features are only visible to users with admin roles.
