
# 🌊 AquaScope User Manual

---

## 📱 Mobile App

### ✨ Overview
AquaScope’s mobile app empowers you to analyze water quality, track predictions, join the community, and manage your profile—all from your device. Navigate easily using the bottom tab bar.

### 🏆 Main Features

| Feature             | Description                                                                                   |
|---------------------|-----------------------------------------------------------------------------------------------|
| **🏠 Home**         | Quick access to water analysis and recent activity.                                           |
| **📝 Data Input**   | Enter water sample data manually or use your camera (OCR). Submit samples for instant analysis.|
| **🧪 Container Analysis** | Scan containers, view/manage scan history, analyze results, and assess risk levels.      |
| **📊 Analytics**    | Visualize water quality trends with interactive charts. Track history and risk assessments.    |
| **📂 Prediction History** | Review all previous water sample predictions with detailed results.                      |
| **💬 Community Forum** | Join discussions, ask questions, share results, and get notifications.                     |
| **👤 Profile**      | Manage your profile, avatar, password, and personal info.                                     |

**🔗 Navigation:**  
Switch between features using the bottom tab bar. Some screens (like Data Input and Profile) are also accessible via navigation actions.

## 💻 Web App (`web-react`)

### 👥 For Users

| Feature         | Description                                                                                   |
|-----------------|-----------------------------------------------------------------------------------------------|
| **🏠 Home**     | Welcome page with project overview and navigation.                                            |
| **📊 Dashboard**| Central hub for scans, analytics, and community features.                                     |
| **🔍 Scans**    | View all water sample predictions linked to your account. Access detailed sample history.      |
| **📈 Analytics**| Visualize water quality data and trends. Export analytics as PDF.                             |
| **💬 Community**| Join discussions, post questions, interact with users, and receive forum notifications.        |
| **👤 Profile**  | Manage your account details, avatar, and password.                                            |

### 🛠️ For Admins

| Feature                | Description                                                                           |
|------------------------|---------------------------------------------------------------------------------------|
| **📊 Admin Dashboard** | Overview of system status and recent activity.                                        |
| **👥 Users Management**| View, search, and manage user accounts. Deactivate users and send notifications.      |
| **📑 Reports**         | Review and manage user reports and flagged content.                                   |
| **🗂️ Forum Management**| Moderate forum posts and threads. Manage categories and user interactions.            |
| **📈 Analytics (Admin)**| Access advanced analytics, export data, and monitor system-wide trends and usage.     |

> **🔒 Note:**  
> All features require authentication. Admin features are visible only to users with admin roles.
