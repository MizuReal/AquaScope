

# 🌊 Aquascope

Aquascope is a multi-platform system for aquatic data analysis, prediction, and visualization. It combines:

- **Mobile App (React Native):** Scan, input, and analyze water samples on the go. View predictions, trends, and join the community forum.
- **Web App (React + Vite):** Full-featured dashboard for users and admins. Visualize analytics, manage scans, moderate forums, and more.
- **Backend (FastAPI, Python):** High-performance API for data processing, AI inference, and integration with Supabase.
- **AI Integration:** Uses GROQ AI for advanced water quality interpretation and contextual feedback.
- **Database:** Supabase for secure, scalable storage and authentication.

- **Database:** Ensure Supabase is configured and credentials are set in the backend `.env` file.
- **Model Files:** Place moss classification model files in the `/model` directory as required.
